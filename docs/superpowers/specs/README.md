# Specs Superpowers — Nexora

Ce dossier contient les spécifications design générées par le skill `brainstorming`.

Chaque spec est créée avant l'implémentation d'une feature, validée par l'utilisateur, puis utilisée par le skill `writing-plans` pour générer le plan d'implémentation correspondant.

## Format des Fichiers

`YYYY-MM-DD-<sujet>-design.md`
